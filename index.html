API Documentation

This API stores incoming data in CSV files. It supports GET and POST requests.

Endpoint

/index.php

Methods

  • GET: sends data via query parameters.
  • POST: sends data as JSON body with Content-Type: application/json.

Student CSV Routing

If you include the query parameter student, data is saved in a student-specific file:

  • Without student: data.csv
  • With ?student=max: data_max.csv

Examples

GET example:

GET /index.php?student=anna&task=1&points=9

POST example:

POST /index.php?student=tom
Content-Type: application/json

{
  "task": 2,
  "points": 7
}

Response

The API returns a JSON object with:

  • success (boolean)
  • message (text)
  • columns (CSV header columns)
